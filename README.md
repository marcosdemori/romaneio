# 🧾 Sistema de Gestão de Romaneio

Sistema web completo para emissão e controle de romaneios, incluindo gestão de clientes, materiais e histórico de movimentações.

---

## 🌐 Acesse online

🔗 https://marcosdemori.github.io/romaneio/

---

## 🚀 Funcionalidades

* 📄 Emissão de romaneios com cálculo automático
* 📊 Controle de área (m²) e valores
* 👤 Cadastro de clientes
* 🪨 Cadastro de materiais
* ⚙️ Cadastro de industrializações
* 🔐 Controle de usuários
* 🧾 Histórico completo de romaneios
* 📤 Exportação de dados em CSV
* 🖨️ Geração de impressão (PDF)
* 📱 Layout responsivo (desktop e mobile)
* 🔗 QR Code com informações de contato

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Supabase (Banco de dados)

---

## ⚙️ Configuração

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

### 2. Configure o Supabase

Crie as seguintes tabelas no banco:

* `romaneios`
* `romaneio_itens`
* `clientes`
* `materiais`
* `industrializacoes`
* `usuarios_sistema`
* `config_empresa`

### 3. Configure a conexão

Ajuste a conexão com o Supabase no arquivo JavaScript (`sb`).

---

## ▶️ Como utilizar

1. Acesse o sistema
2. Realize o login
3. Cadastre clientes e materiais
4. Crie um novo romaneio
5. Finalize para gerar e salvar
